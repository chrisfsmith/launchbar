## UUID
[LaunchBar](https://www.obdev.at/products/launchbar/index.html) extension to generate a UUID.

## Usage Examples:
* To generate a UUID
    1. Find the action by typing **UUID** in LaunchBar
    2. `SPACE` or `ENTER` to generate the UUID
